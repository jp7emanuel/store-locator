# Jp7 Store Locator

#### Introduction

A ReactJS app which uses google maps services to search for user nearest stores.

#### Feats

* Express
* MongoDB with Mongoose
* React + Redux
* Google Maps Services (directions and location)
* Bulma CSS
* Font Awesome
* Gulp
* Redux Form with validations
* Alerts using SweetAlert
* ES6 everywhere!
